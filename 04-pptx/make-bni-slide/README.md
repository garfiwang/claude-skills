# 製作 BNI 簡報 (make-bni-slide)

> 專為 BNI（Business Network International）會員設計的自動化簡報生成技能。支援 PPTX 及 HTML 互動網頁簡報雙格式，完美契合 BNI 官方配色與品牌視覺規範。

---

## 🌟 技能特色與功能

### 1. 雙格式支援
*   **經典 PPTX 簡報**：適合下載後在 Microsoft PowerPoint 或 Keynote 中進行細部調整與報告。
*   **互動 HTML 簡報**：由 Reveal.js 強力驅動，具備豐富的動態轉場、響應式佈局、動態卡片網格與脈動式視覺提醒，適合線上即時展示。

### 2. 官方視覺規範（完美對比與易讀性）
*   **精準品牌色**：
    *   **主色 (BNI 紅)**：`#d12031`（大標題與關鍵焦點）。
    *   **次色/內文 (炭灰深藍)**：`#1a2530`（極佳的閱讀體驗，不易疲勞）。
    *   **強調色 (金黃/琥珀)**：`#b67b12`（點綴關鍵提示）。
*   **官方底圖整合**：自動套用包含左下角 BNI Logo 與右下角金紅弧線的 `bnilogo.png`。文字內容會自動避開邊界標誌，在大螢幕或投影幕上皆能完美呈現。
*   **自動防止折字**：封面字級經精心計算（`1.5em !important`），避免中文字串被突兀截斷或單個字落單換行。

### 3. 高質感內容排版
*   **結構化封面頁**。
*   **EDM 資訊頁**：使用卡片網格 (`display: grid`)，搭配精美卡片陰影與脈動（Pulsing）外框警示（如截止日期等）。
*   **預演與登記頁**：待辦事項清單樣式，附有醒目的 BNI 紅色實心「線上登記/預演按鈕」，一鍵點擊即可直達目的地。

### 4. 開發者友好（自動部署）
*   HTML 簡報產出後，支援一鍵開啟本地伺服器 (`http://localhost:8000`) 進行預覽。
*   使用者確認滿意後，可使用 GitHub CLI 一鍵在 GitHub 建立倉庫並發佈至 GitHub Pages，自動產生線上公開的簡報連結。

## 📦 安裝方式

在您的 **Claude Code** 終端機中執行以下命令即可安裝此技能：

```bash
claude skills add garfiwang/claude-skills --path 04-pptx/make-bni-slide
```
