# Notion 靈感收集資料盒

## 資料庫資訊

- **Data Source ID**：`32ac9c67-1f57-81b8-91b0-000b1d592084`

## 資料庫欄位

| 欄位 | 類型 | 可用選項 |
|------|------|----------|
| 標題 | 文字（必填） | — |
| 內容 | 文字 | — |
| 分類 | 單選 | 想法、靈感、資訊、待辦、其他、笑話 |
| 標籤 | 多選 | 設計、寫作、商業、生活、技術、演講 |
| 來源 | 文字 | — |
| 建立時間 | 自動 | — |

## 儲存流程

1. 從使用者訊息中擷取：標題、內容、分類、標籤、來源
2. 若使用者未指定分類，根據內容語意自動推斷最合適的分類
3. 若使用者未指定標籤，根據內容自動推斷（可不填）
4. 呼叫 `Notion:notion-create-pages` 儲存到資料庫
5. 回覆簡短確認，附上 Notion 頁面連結

## 呼叫範例

```
parent: { type: "data_source_id", data_source_id: "32ac9c67-1f57-81b8-91b0-000b1d592084" }
pages: [{
  icon: "💡",
  properties: {
    "標題": "<標題>",
    "內容": "<內容>",
    "分類": "<分類>",
    "標籤": '["<標籤1>", "<標籤2>"]',
    "來源": "<來源>"
  }
}]
```

## 注意事項

- 標題若使用者沒給，從內容中自動生成一個簡短標題
- 分類只能選其中一個；標籤可以多個
- 儲存成功後回覆要簡短，不要冗長說明
