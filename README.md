## 安裝方式

在 Claude Code 中執行：

```bash
# 安裝單一 skill（以 storytelling-7steps 為例）
claude skills add garfiwang/claude-skills --path 01-content-creation/storytelling-7steps
```

或直接複製 `SKILL.md` 到你的 Claude Code 工作目錄下使用。
